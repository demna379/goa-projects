JavaScript Loops

What Are Loops?

Loops in JavaScript are programming constructs that allow you to execute a block of code repeatedly as long as a certain condition is true. Instead of writing repetitive code, you can use loops to handle tasks like iterating over arrays, repeating actions, or automating tasks efficiently.

History of Loops in JavaScript

Loops have been part of JavaScript since its creation in 1995. Inspired by the C programming language, JavaScript adopted traditional looping mechanisms like for, while, and do...while. Later, as the language evolved, new types of loops were added such as for...in (to iterate over object properties) and for...of (introduced in ES6 in 2015 for iterating over iterable objects like arrays and strings).

Why Are Loops Useful?

  • Repetition: Run the same block of code multiple times without duplication.
  • Efficiency: Save time and reduce errors by automating tasks.
  • Scalability: Easily handle large sets of data without manually coding each step.
  • Flexibility: Different types of loops exist to fit specific use cases.

Types of Loops in JavaScript

1. For Loop

The for loop is the most common loop in JavaScript. It repeats code a specific number of times. 

for (let i = 0; i < 5; i++) {
  console.log("Number: " + i);
}

This loop will output numbers 0 through 4.

2. While Loop

The while loop keeps running as long as a condition is true. 

let i = 0;
while (i < 5) {
  console.log("Number: " + i);
  i++;
}

This loop works similarly to the for loop but uses a condition directly.

3. Do...While Loop

The do...while loop runs the code at least once before checking the condition. 

let i = 0;
do {
  console.log("Number: " + i);
  i++;
} while (i < 5);

Even if the condition is false initially, the code runs once.

4. For...In Loop

The for...in loop iterates over the keys/properties of an object. 

const person = {name: "Alex", age: 25, job: "Developer"};

for (let key in person) {
  console.log(key + ": " + person[key]);
}

This will print each property name and its value.

5. For...Of Loop

The for...of loop iterates over iterable objects such as arrays, strings, and more. 

const colors = ["red", "green", "blue"];

for (let color of colors) {
  console.log(color);
}

This will output each color in the array.

Break and Continue in Loops

Loops can be controlled with break (to stop the loop completely) and continue (to skip the current iteration and move to the next one). 

for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}

This will print numbers 0 through 4 and stop once it hits 5.

for (let i = 0; i < 10; i++) {
  if (i % 2 === 0) continue;
  console.log(i);
}

This will only print odd numbers because continue skips even ones.

Nested Loops

You can put loops inside loops. This is useful for handling multi-dimensional arrays or complex tasks. 

for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log("i=" + i + ", j=" + j);
  }
}

This will print all combinations of i and j from 1 to 3.

Practical Examples of Loops

Example 1: Loop Through an Array

const fruits = ["apple", "banana", "cherry"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Example 2: Loop Through a String

const word = "HELLO";

for (let char of word) {
  console.log(char);
}

Example 3: Sum of Numbers with a Loop

let sum = 0;
for (let i = 1; i <= 5; i++) {
  sum += i;
}
console.log("Sum = " + sum); // Output: 15
