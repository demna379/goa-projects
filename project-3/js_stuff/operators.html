Javascript operators

Arithmetic Operators

Arithmetic operators let you do math in JavaScript. Common ones are:

  • + : addition
  • - : subtraction
  • * : multiplication
  • / : division
  • % : remainder (modulus)

Use them to calculate values, totals, or perform numeric operations.

Comparison & Logical Operators

Comparison operators let you compare values, returning true or false:

  • == : equal to
  • === : equal value and type
  • != : not equal
  • < : less than
  • <= : less than or equal
  • > : greater than
  • >= : greater than or equal

Logical operators are used to combine conditions:

  • && : AND
  • || : OR
  • ! : NOT

They help make decisions in if statements and loops.
