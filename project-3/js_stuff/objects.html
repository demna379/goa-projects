Understanding JavaScript Objects

What is an Object?

In JavaScript, an object is a collection of keyâ€“value pairs. Objects allow you to store multiple related pieces of data together in a single structure. 


let person = {
  name: "Alex",
  age: 25,
  job: "Developer"
};

Accessing Object Properties

You can access values using dot notation or bracket notation.


console.log(person.name); // Alex
console.log(person["age"]); // 25

Changing Object Properties


person.age = 26; // change age
person["job"] = "Designer"; // change job
console.log(person);
// { name: "Alex", age: 26, job: "Designer" }

Adding and Removing Properties


// Add a new property
person.city = "New York";

// Remove a property
delete person.job;

console.log(person);
// { name: "Alex", age: 26, city: "New York" }

Object Methods

Objects can also contain functions called methods.


let user = {
  name: "Maya",
  greet: function() {
    return "Hello, " + this.name + "!";
  }
};

console.log(user.greet()); // Hello, Maya!

Looping Through Objects

The for...in loop lets you iterate over properties.


for (let key in person) {
  console.log(key + ": " + person[key]);
}
// name: Alex
// age: 26
// city: New York

Objects with Arrays


let student = {
  name: "John",
  grades: [90, 85, 78],
  average: function() {
    let sum = 0;
    for (let g of this.grades) {
      sum += g;
    }
    return sum / this.grades.length;
  }
};

console.log(student.average()); // 84.333...

Nested Objects

Objects can contain other objects as properties.


let company = {
  name: "TechCorp",
  address: {
    city: "San Francisco",
    zip: "94105"
  }
};

console.log(company.address.city); // San Francisco

Object.keys() and Object.values()


console.log(Object.keys(person)); // ["name", "age", "city"]
console.log(Object.values(person)); // ["Alex", 26, "New York"]
