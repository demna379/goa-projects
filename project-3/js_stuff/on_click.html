About JavaScript onclick

The onclick event in JavaScript lets you run code when a user clicks on an element (like a button, image, or even plain text).

It was introduced in the early days of the web (around HTML 4 / late 1990s), making it one of the oldest and most widely used event handlers.

Why it’s useful: it allows websites to react to user actions. For example, you can use it to open a popup, toggle a menu, change text, or run any custom function.

Example: 

<button onclick="alert('You clicked me!')">Click me</button>

These days, developers often use addEventListener('click', ...) instead of onclick because it’s more flexible, but onclick still works perfectly fine and is easy to use for beginners.
