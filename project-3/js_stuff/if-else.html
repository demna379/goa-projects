JavaScript if...else Statement

The if...else statement in JavaScript lets you make decisions in your code. It checks a condition (something that can be true or false) and runs different blocks of code depending on the result.

How it works:

  • if – runs the code inside if the condition is true.
  • else – runs if the condition is false.
  • else if – lets you check multiple conditions in order.

Why it’s useful:

It allows your program to react to different situations. For example, checking if a user is logged in, verifying input, or choosing different paths in a game.

Fun fact:

Conditional statements like if...else have been part of JavaScript since its very first version in 1995. They’re borrowed from older programming languages like C and Java, so if you learn it in JS, you’ll recognize it elsewhere too.
