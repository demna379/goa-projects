JavaScript and the DOM

What is the DOM?

The DOM (Document Object Model) is a representation of the HTML structure of a webpage. It allows JavaScript to read and manipulate content, structure, and style.

Elements in the DOM

The DOM represents every HTML element as a node that can be accessed and changed.


<h1>Hello</h1> 
<p>This is a paragraph.</p>

Each of these elements is a node in the DOM tree.

DOM Tree

The DOM is structured like a tree: the <html> element is the root, <head> and <body> are branches, and all other elements are child nodes.

Attributes

Elements can have attributes, which provide extra information.


<img src="image.jpg" alt="A picture">
<a href="https://example.com">Link</a>

Classes and IDs

Classes and IDs are used to identify elements in the DOM. IDs should be unique, while multiple elements can share a class. 


<div id="header">Header</div>
<div class="menu">Item 1</div>
<div class="menu">Item 2</div>

Nested Elements

Elements can be inside other elements, forming a nested structure.


<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Text Nodes

Even the text inside elements is considered a node in the DOM.


<p>Hello, world!</p>

Here "Hello, world!" is a text node inside the <p> element.

Why the DOM is Important

JavaScript interacts with the DOM to make webpages dynamic. You can:

  • Change content
  • Modify attributes
  • Insert or remove elements
  • React to user actions

Common DOM Elements

Here are some examples of elements you'll find in the DOM:

  • <div>, <span>, <p>, <h1> to <h6>
  • <ul>, <ol>, <li>
  • <img>, <a>
  • <input>, <form>, <button>

Summary

The DOM is a powerful way to represent HTML so that scripts can interact with web pages. Understanding it is key to manipulating content, handling events, and making your site interactive.
