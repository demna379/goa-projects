Understanding JavaScript Variables

What is a Variable?

A variable is a container that stores data values. In JavaScript, variables can hold numbers, text, arrays, objects, or even functions.

Declaring Variables

You can declare variables using var, let, or const.


var name = "Alex";    // old way, function-scoped
let age = 25;         // modern way, block-scoped
const pi = 3.14159;   // constant, cannot be changed

Variable Naming Rules

  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must start with a letter, underscore (_), or dollar sign ($).
  • Names cannot be reserved keywords like if, function, or var.
  • JavaScript is case-sensitive: age and Age are different variables.

Assigning Values

You can assign or reassign values to variables after declaring them.


let score = 10;
score = 20; // value changed

Types of Variables

Variables can store different types of data:

  • String: text, e.g., let name = "Alex";
  • Number: integers or decimals, e.g., let age = 25;
  • Boolean: true or false, e.g., let isOnline = true;
  • Array: list of values, e.g., let fruits = ["Apple","Banana"];
  • Object: key-value pairs, e.g., let person = {name:"Alex", age:25};
  • Undefined & Null: special empty values

Why Variables Are Useful

Variables let you store and reuse data throughout your program. You can perform calculations, track information, and dynamically change content.

Example of Variables in Action


let userName = "Maya";
let userAge = 30;
let isMember = true;

console.log("Name:", userName);
console.log("Age:", userAge);
console.log("Member:", isMember);

Summary

Variables are the building blocks of JavaScript. Understanding how to declare, assign, and use them is essential for any interactive web page or application.
