Understanding JavaScript Arrays

What is an Array?

An array in JavaScript is a special variable that can hold multiple values at once. Instead of creating multiple variables, you can store them inside an array. 


// Example of an array
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits); // ["Apple", "Banana", "Cherry"]

Accessing Array Elements

Arrays use indexes, starting at 0, to access elements.


let colors = ["Red", "Green", "Blue"];
console.log(colors[0]); // Red
console.log(colors[1]); // Green
console.log(colors[2]); // Blue

Changing Array Elements

You can replace an element using its index.


let numbers = [1, 2, 3, 4];
numbers[2] = 99;
console.log(numbers); // [1, 2, 99, 4]

Array Properties

The length property shows how many items are in the array.


let animals = ["Dog", "Cat", "Elephant"];
console.log(animals.length); // 3

Array Methods

1. push()

Adds an item at the end.


let games = ["Minecraft", "Roblox"];
games.push("Valorant");
console.log(games); // ["Minecraft", "Roblox", "Valorant"]

2. pop()

Removes the last item.


let cities = ["Tokyo", "Paris", "New York"];
cities.pop();
console.log(cities); // ["Tokyo", "Paris"]

3. shift()

Removes the first item.


let cars = ["BMW", "Tesla", "Audi"];
cars.shift();
console.log(cars); // ["Tesla", "Audi"]

4. unshift()

Adds an item at the beginning.


let drinks = ["Water", "Juice"];
drinks.unshift("Milk");
console.log(drinks); // ["Milk", "Water", "Juice"]

Looping Through Arrays

Arrays are often used with loops.


let sports = ["Football", "Basketball", "Tennis"];

for (let i = 0; i < sports.length; i++) {
  console.log(sports[i]);
}
// Football
// Basketball
// Tennis

Array with forEach()

The forEach() method runs a function for each element.


let numbers = [10, 20, 30];
numbers.forEach(function(num) {
  console.log(num * 2);
});
// 20
// 40
// 60

Multi-dimensional Arrays

You can also create arrays inside arrays.


let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][2]); // 6
