Who Created It

HTML was created by Tim Berners-Lee in 1991 while working at CERN. It was initially designed to share documents and information over the internet

What It Does

HTML structures web content and allows browsers to display text, images, and multimedia.

How to Use It

HTML is used to create web pages by combining elements like headings, paragraphs, and links.

HTML tags

in this section you will learn what tags are. HTML tags are the building blocks of an HTML document. They are used to define and structure the content of a webpage. Tags are enclosed in angle brackets and usually come in pairs: an opening tag and a closing tag. The content between these tags is the element's content.

  1. Syntax:

    2. Tags are written as for the opening tag and for the closing tag.

  2. Self-Closing Tags:

    3. Some tags do not require a closing tag and are self-closing.

  3. Attributes

    4. Tags can have attributes that provide additional information about the element. Attributes are written inside the opening tag.

  4. Common Tags:

    5. h1 to h6: heading

    p: paragraph

    a: anchor (link)

    img: image

    div: division (container)

    span: inline counter

    ul and ol: unordered and ordered lists

    table: table

  5. Semantic Tags:
    6. Modern HTML includes semantic tags that describe the purpose of the content, such as:

    header: Defines a header section.

    footer: Defines a footer section.

    article: Represents an independent piece of content.

    section: Groups related content.
